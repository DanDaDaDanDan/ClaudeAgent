# Image Generation Prompts v2 - "The Tiny World"

## Global Style Guide

**Apply to ALL prompts:**
```
Style: hyperrealistic fantasy, cinematic lighting, photorealistic textures,
8k ultra detailed, dramatic scale contrast, vertical 9:16 composition,
professional photography, shallow depth of field where appropriate,
film grain subtle, atmospheric
```

**Negative prompts (if supported):**
```
cartoon, anime, low quality, blurry, text, watermark, oversaturated,
plastic looking, flat lighting, amateur
```

## Consistency Checklist

Before generating all images, verify:
- [ ] All use 9:16 vertical aspect ratio
- [ ] Bioluminescent colors consistent (cyan/amber/gold)
- [ ] Tiny beings have consistent ambiguous-humanoid design
- [ ] Mushroom architecture style maintained
- [ ] Surface world is muted/realistic, underground is magical/glowing
- [ ] Scale relationship maintained (shoe = massive, beings = tiny)
- [ ] Space left for text overlays in Shot 1 start and Shot 3 end

---

## Reference Image Strategy

**Recommended approach:**
1. Generate Shot 1 End Frame (city aerial) first - this establishes the world
2. Use that as reference for all underground shots
3. Generate Shot 1 Start Frame (shoe) separately - different style (realistic)
4. Generate Shot 3 End Frame last - bridges both styles

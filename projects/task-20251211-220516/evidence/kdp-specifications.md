# KDP Comic Book Specifications

## Recommended Trim Sizes for Comics/Graphic Novels

| Size | Use Case |
|------|----------|
| **6.625" x 10.25"** | Standard American comic size |
| **6" x 9"** | Versatile, professional look |
| **5.5" x 8.5"** | Compact graphic novel |
| **8.5" x 11"** | Magazine/art book format |

**Recommended for this project**: 6.625" x 10.25" (industry standard comic) OR 6" x 9" (trade paperback graphic novel)

## Bleed Requirements

- **Bleed extension**: 0.125" (3.2mm) beyond trim on all sides
- **Safe zone**: Keep critical text/elements 0.25" from trim edge
- If ANY page has bleed, entire file must be set up with bleed

## Page Count Standards

- **Standard comic book**: 24-32 pages
- **Short story collection**: Can be 48-64 pages for two stories
- **Page count must be divisible by 2** (front and back)

## Panel Conventions

### American Style (Our Target)
| Metric | Range | Ideal |
|--------|-------|-------|
| Panels per page | 3-12 | **5-6** |
| Words per panel | 0-35 | **25 or less** |
| Words per page | 0-200 | **150 or less** |
| Story points per page | 1-4 | **2-3** |

### Special Panel Types
- **Splash page**: Full-page single panel (impactful moments)
- **Double-page spread**: Two pages, one image (major reveals)
- **Silent panels**: No text, pure visual storytelling
- **Grid layouts**: Regular grid (e.g., 9-panel) for rhythm/tension

## Story Length Calculation

For a **compelling short story** (~12-16 pages per story):
- At 5 panels/page average: **60-80 panels per story**
- Total for 2 stories: **120-160 panels**
- Plus title pages, credits: ~32-36 pages total

## File Requirements

| Specification | Requirement |
|---------------|-------------|
| Resolution | 300 DPI minimum, 600 DPI max recommended |
| File size | Under 650MB |
| Format | Single pages (not spreads) |
| Fonts | Embedded |
| Color | Premium color or standard color option |

## Our Project Specifications

**Format**: 6" x 9" trade paperback graphic novel
**Page count**: 36 pages total
- Cover (not counted in interior)
- Title page (1)
- Story 1 title (1) + Story (14 pages)
- Story 2 title (1) + Story (14 pages)
- Credits/back matter (5 pages)

**Panels per story**: ~70 panels (5 panels/page average)
**Total panels**: ~140 panels

**Image output**: 1800 x 2700 pixels at 300 DPI (6" x 9")
